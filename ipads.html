iPad models & specifications


iPad Pro 13-inch (M4, 8th generation, 2024) (Models with 1TB/2TB storage)

Hardware

  • Apple M4 chip - From 2024
  • 10-core CPU with 4 performance cores and 6 efficiency cores
  • 10-core GPU
  • 16 GB memory
  • 256 GB, 512 GB, 1 TB, or 2 TB storage
  • 16-core Neural Engine
  • Hardware accelerated ray-tracing support
  • Media engine with 8K ProRes encode/decode and AV1 decode

Display

  • 13-inch display size
  • Tandem OLED "Ultra Retina XDR" display
  • Edge to edge design with rounded corners
  • 2752-by-2064 resolution at 264 ppi
  • 120 Hz ProMotion
  • 1000 nits SDR brightness
  • 1600 nits peak brightness

WiFi, Bluetooth, Ports

  • WiFi 6E
  • Bluetooth 5.3
  • USB-C charging port
  • Thunderbolt / USB 4 support, up to 20Gbps transfer

Operating System

  • Minimum OS: iPadOS 18
  • Newest OS: Current (iPadOS 26)
  • Currently supported

If you cannot run Vulkan, update your graphics drivers or try a different computer.

Important for macOS users

  • Boiby App is designed to run on both Intel (x86) Macs and Apple Silicon (M1 chip and newer) Macs.
  • You will need to allow the app through gatekeeper, as Apple blocks it as it's not a signed application. You can do this through System Settings.
  • A Metal-capable GPU is required to run this app. This means most Mac models from 2012 and later that support macOS Catalina 10.15 natively. Mac Pro models from 2008-2012, and iMac models from late 2009 until 2011 can be upgraded with a Metal-capable GPU. Older Mac models that do not support macOS 11 Big Sur can be upgraded to it for free using the OpenCore Legacy Patcher
