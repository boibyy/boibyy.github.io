Install or Upgrade Memory in Your Boiby Pro (2025)

Updated October 9, 2025

Memory Compatibility

  • Supported memory types: 5600 MHz DDR5 ECC Registered DIMMs (R-DIMM) or Load-Reduced DIMMs (LR-DIMM).
  • Do not mix R-DIMMs and LR-DIMMs; doing so will trigger a memory error flash on the status indicator light.
  • DIMMs with heatsinks are not supported.
  • 12 DIMM slots available on six-channel memory controller:
    • BX6 Ultra chip: up to 1.5 TB memory.
    • BX6 Extreme chip: up to 3 TB memory.
  • Supported configurations: 4 or 8 slots for quad-channel, 6 or 12 slots for six-channel, 10 slots for mixed operation.

Recommended Practices

  • Use identical memory modules across all slots to maximize performance.
  • If using R-DIMMs of mixed capacities:
    • Use only R-DIMMs.
    • Install larger DIMMs in the first slot of each channel pair.
  • If using LR-DIMMs, all DIMMs must be identical in capacity.

Step 1: Remove the Top Cover

  • Shut down your Boiby Pro and wait 5 minutes to cool down.
  • Unplug all cables except the power cord.
  • Discharge static by touching the metal housing, then unplug the power cord.
  • Flip the top latch up, twist left to unlock, and lift the housing straight up.

Step 2: Remove DIMMs

  • Slide each lock left to open the DIMM covers.
  • Locate the DIMM(s) to remove. Push down the ejectors on each end; the DIMM will lift slightly.
  • Grip the DIMM at the ends and pull straight out; do not twist or force.
  • Place removed DIMMs in an ESD-protective bag.

Step 3: Install DIMMs

  • Locate an empty DIMM slot.
  • Align the DIMM with the slot and insert gently, and ensure the notch is aligned.
  • Press until the bottom edge touches the gold contacts, then fully seat the DIMM until the ejectors click.
  • Replace DIMM covers, pressing down until they click.

Step 4: Reinstall the Top Cover

  • Lower the housing over the Boiby Pro.
  • Twist the top latch right and flip down to lock.
  • Reconnect power, display, and peripherals.

Other Configurations

  • Test a single DIMM by installing it in the first slot of a channel.
  • Test the second slot of a channel by installing DIMMs in both slots of that channel.

Tips

  • Always discharge static before touching internal components.
  • Do not mix R-DIMMs and LR-DIMMs.
  • Do not use DIMMs with heatsinks; they can damage the slot.
  • Follow supported memory configurations to avoid boot errors.
