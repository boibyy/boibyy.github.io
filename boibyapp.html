The Boiby App

What is the Boiby App?

Boiby is a virtual desktop cat that lives inside a window on your computer. Built in Godot, Boiby responds to simple keyboard controls: press Space to hear a meow, and E to change Boiby's image. More features coming soon.

The project is currently in beta (version beta 3.0) and available for Windows, macOS and Linux.

Controls & Features

  • Space - play one of Boiby's meows (6 meows in this beta).
  • E - cycle Boiby's images (2 images in this beta).
  • (Coming soon) Resizable windowed mode; planned settings and preferences.
More meows, images and features are coming soon.

Supported Platforms

Boiby beta 3.0 runs on:

  • Windows 10 (64-bit) and later
  • macOS 11 Big Sur and later
  • Modern 64-bit Linux distributions

A GPU that supports Vulkan rendering is required for Godot's renderer to function properly. See the download page for full details and a list of available builds.

About the Beta

This is an experimental build. Expect occasional crashes, missing features, and issues. Please report bugs and feedback via the contact page.

Current build: beta 3.0
