Replace the LPCAMM2 Memory Module

Updated October 9, 2025

Memory Compatibility

  • BoibyBook Pro 14-inch B5 (2024) & B6 (2025): 16 GB, 24 GB, or 32 GB LPDDR5-7500 LPCAMM2 (one module supported).
  • BoibyBook Pro 14-inch & 16-inch B5 Pro (2024) & B6 Pro (2025): 32 GB, 48 GB, or 64 GB LPDDR5X-8533 LPCAMM2 (one module supported).
  • BoibyBook Pro 14-inch & 16-inch B5 Max (2024) & B6 Max (2025): 48 GB, 64 GB, 96 GB, or 128 GB LPDDR5X-8533 LPCAMM2 (two modules supported; both modules must match capacity and timings, and be fully populated, or the computer will not boot).

Before You Begin

  • Check your BoibyBook Pro model and memory compatibility.
  • Back up your files using Boiby Time or another backup method.
  • Gather tools: Torx screwdriver (T5 or T6 depending on model), spudger, and optional anti-static wrist strap.

Step 1: Shut Down and Unplug

Power off your BoibyBook Pro completely and disconnect all cables, including the power adapter.

Step 2: Remove the Bottom Cover

Flip the BoibyBook Pro over, remove all Torx screws securing the bottom cover, and lift the cover carefully using a spudger.

Step 3: Remove the Old Memory Module

  • Locate the LPCAMM2 memory module on the logic board.
  • Push the side clips outward to release the module.
  • Lift the module out at an angle, holding it by the edges.

Step 4: Install the New Memory Module

  • Align the notch on the new module with the socket.
  • Insert at a 30° angle and press until the clips click.
  • For models with two modules (B5 Max/B6 Max), repeat for the second module ensuring both have identical capacity and timings.

Step 5: Reassemble and Check

  • Replace the bottom cover and secure all Torx screws.
  • Reconnect power and peripherals.
  • Power on the BoibyBook Pro and verify the new memory is recognized under About This BoibyBook → Memory.

Tips

  • Use Boiby-certified LPCAMM2 modules for best performance.
  • Handle all components carefully to prevent static damage.
  • Memory only fits one way—do not force it into the socket.
